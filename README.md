`npm install unnamed`

# Development

# For web apps

Npm

`npm install unnamed`

Usage with webpack: `import 'unnamed'`

# For landing pages

<h3>Clone</h3>

`git clone https://github.com/smakosh/Unnamed-getting-started.git`

`cd Unnamed-getting-started && yarn install && bower install`

`gulp serve`

<h3>Build</h3>

`gulp build`
